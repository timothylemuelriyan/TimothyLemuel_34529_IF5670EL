TextEditor dan Storage
-user memasukan nama file kemudia klik "open",jika file tersebut belum tersimpan, maka
akan muncul popup "file tidak ditemukan". jika file sudah disimpan, drop down list dari seluruh file yang tersimpan.
-save = berfungsi untuk menyimpan file baru yang sudah dibuat
-delete = berfungsi untuk menghapus file yang dibuat
-clear = membersihkan file / mengkosongkan file yang sedang di buat
-exit = keluar dari aplikasi
